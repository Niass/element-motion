# Move

Useful for animating an element moving from one position to another,
works best when the element has **very little visual difference**.
This animation uses the [FLIP technique](https://css-tricks.com/animating-layouts-with-the-flip-technique/) so it is quite performant.

## Usage

```js
import { Move } from 'yubaba';
```

<Playground>
  <Common.Toggler interval>
    {({ shown, toggle }) =>
      shown ? (
        <Animator name="move" key="1">
          <Move>
            {({ ref, style }) => <Styled.Box onClick={() => toggle()} style={style} ref={ref} />}
          </Move>
        </Animator>
      ) : (
        <Animator name="move" key="2">
          <Move>
            {({ ref, style }) => (
              <Styled.Box style={style} ref={ref} onClick={() => toggle()} alignRight />
            )}
          </Move>
        </Animator>
      )
    }
  </Common.Toggler>
</Playground>
