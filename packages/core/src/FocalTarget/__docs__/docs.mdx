# FocalTarget

Used to mark the focal element,
only a handful of motions require this component to be used,
for example [FocalConcealMove](/focal-conceal-move).

> **Tip -** See [Moving using a focal target](/advanced-usage#moving-using-a-focal-target) for more information on how to use this component.

## Usage

```js
import { FocalTarget, FocalConcealMove } from '@element-motion/core';

const ItemDetails = ({ title, recipients, body, index }) => (
  <Motion name={`item-${index}`}>
    <FocalConcealMove>
      {motion => (
        <div {...motion}>
          <FocalTarget>
            {({ ref }) => (
              <div ref={ref}>
                {title}
                {recipients}
              </div>
            )}
          </FocalTarget>

          {body}
        </div>
      )}
    </FocalConcealMove>
  </Motion>
);
```
